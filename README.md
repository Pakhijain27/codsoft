Projects for Codsoft

This repository contains the projects of python developed during my intership at Codsoft. Differnt projects that I have made during internships focuses on Python programming, such as games, contact details, password generator and more.

Projects

Project 1: Rock, Paper, Scissors

This Python game allows users to play Rock Paper Scissors against the computer. The computer randomly selects an option, and the game determines the winner based on the classic rules of Rock Paper Scissors.

Project 2: Calculator

This application is a versatile command-line tool that performs basic arithmetic operations, including addition, subtraction, multiplication, and division.

Project 3: Contact Book

A Contact Book application where users can add, view, update, and delete contact information. 

Project 4: Password Generator

This application generates the strong and random passwords for users. It allows users to specify the length and complexity of the password.
